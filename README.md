## Installation

<ol>
	<li>Download the dylib and whitelist files from the latest release <b><a href="https://github.com/ChristopherA8/LogPaper/releases/latest">here</a></b></li>
	<li>Move the dylib and whitelist files into <code>/var/ammonia/core/tweaks</code></li>
	<li>Run <code>killall Finder</code></li>
</ol>

## Uninstalling

<ol>
	<li>Download the <code>Makefile</code> from this repo and run <code>make uninstall</code></li>
</ol>
